description: Manage git push, pull, and sync with GitHub. Three modes — push (end of session), pull (start of session), full sync (pull then push).
---

Read the full skill documentation at `skills/git-sync.md` before proceeding.

Then ask the user which mode they need:
- **Push** — stage, commit, and push today's changes to GitHub
- **Pull** — pull latest from remote and show what changed
- **Full sync** — stash local changes, pull, re-apply, then push

Once the mode is confirmed, follow the exact prompt and steps defined in `skills/git-sync.md` for that mode.
